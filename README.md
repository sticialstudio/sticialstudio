# STEMAIDE Web Studio

A modern educational platform bringing LMS, block-based coding (Blockly), and text coding (Monaco) together for Arduino-based learning.

Get started:
- Install dependencies: npm install
- Run dev: npm run dev
- Open http://localhost:3000

This is a scaffold. It includes:
- App Router Next.js setup with /courses, /projects, /editor routes
- Prisma models using SQLite for local development
- Endpoints: /api/compile for Arduino code (simulated compile)
- Blockly-based block editor and Monaco-based code editor stubs
